# ISBN 校验与修正程序

这个项目提供了一个命令行工具，用于验证并自动修正用户输入的 ISBN-13 编码。程序会检查格式、计算校验位，并输出验证结果或纠正后的 ISBN。

---

## 项目特点

* 接受用户输入的十三位 ISBN 字符串
* 自动处理输入中的非数字字符错误
* 按顺序为前十二位数字计算校验和
* 自动判断校验位是否正确
* 在校验位错误时输出修正后的完整编码

本程序结构清晰，并通过异常捕获避免运行时错误中断执行。

---

## 如何运行

1. 确保已安装 Python
2. 将程序保存为文件，例如
   `isbn.py`
3. 通过命令行进入该文件所在目录
4. 执行

```
python isbn.py
```

5. 按提示输入十三位 ISBN
6. 查看验证结果或自动修正后的编号

---

## 代码说明

以下是程序核心逻辑的结构化讲解，未包含完整代码，以保持说明简洁。

### 输入读取与基本检查

程序首先读取用户输入并进行长度判断。若长度不是十三位，直接提示错误并结束执行。之后逐位检查输入是否为可转换的数字字符，若遇到非法字符立即提示错误。

### 校验和的计算方式

程序为前十二位数字依次赋予从一到十二的权重，并累计每一位数字与其权重的乘积。最终对累加值取余数，用以得到校验位。

### 校验位验证与修正

若用户输入的第十三位校验位与计算结果一致，则输出 Right。
如果不一致，则根据计算结果替换校验位，重新输出修正后的完整 ISBN。

### 程序结构

主逻辑被封装在一个函数中并由入口保护运行，使得代码更易维护，也便于未来进行扩展。
